# PKU 高数 C 笔记整理

本仓库用于整理和维护北京大学《高等数学 C》课程相关学习笔记，主要内容来自个人手写课堂笔记，并在此基础上进行系统化、电子化和结构化整理。

整理目标不是简单转录，而是将原始笔记改写为更适合复习、查阅和考前回顾的电子版资料。

---

## 仓库内容

目前主要包含：

- 多元函数微分学
- 多元函数、二重极限与连续
- 偏导数与全微分
- 方向导数与梯度
- 复合函数与隐函数微分法
- 高阶偏导数
- 偏导数的几何应用
- 多元函数极值与条件极值
- 二重积分及后续章节整理中

---

## 笔记特点

本仓库中的笔记按照统一结构整理：

1. 核心概念  
2. 公式与结论汇总  
3. 推导过程  
4. 例题整理  
5. 图像或几何意义  
6. 重点与易错点  
7. 复习版总结  

尽量做到：

- 概念表述清楚
- 公式条件明确
- 推导步骤完整
- 易错点单独标出
- 适合期末复习和查漏补缺
